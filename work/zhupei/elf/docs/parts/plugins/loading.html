加载提示插件

Elf.components.loading

options

width Number
默认：64

动画图标的宽度

width Number
默认：64

动画图标的高度

color color
默认：颜色值

loading 动画颜色

target Element
默认：Element Object default document.body

loading 动画颜色

举例:

		
		//创建显示loading
		Elf.components.loading();
		//关闭并销毁loading
		Elf.components.loading("close");
