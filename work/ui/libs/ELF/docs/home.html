确认提示框

Elf.components.confirm

options

width Number
默认： null

宽度

height Number
默认：null

minWidth String
默认：200px

最小宽度

maxWidth String
默认：300px

最大宽度

modle Boolean
默认： true

是否显示遮罩

title String
默认： null

显示标题 可以为空

text String
默认：

提示消息，必须

buttons Object
默认： {"确定":function(){}}

按钮的个数，按钮文字，和点击执行的方法

举例:

		
		Elf.components.confirm({
			title:"温馨提示",
			text:"这是提示信息",
			modle:false,
			buttons:{
				"确定":function(){
					console.info("确定 clicked 1");
				},
				"取消":function(){
					console.info("取消 clicked");
				}
			}
		});
